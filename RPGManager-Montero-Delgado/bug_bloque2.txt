BUG BLOQUE 2 - CORRECCION DE COMPARACION

Problema:
if (clase = "Guerrero") { ... }

Descripcion del error:
En Java el operador = se usa para asignar valores, no para comparar.
Usarlo dentro de un if genera un error logico y de sintaxis en este caso.

Solucion:
Para comparar Strings en Java se debe usar el metodo .equals().

Correccion:
if (clase.equals("Guerrero")) {
    System.out.println("Ataque cuerpo a cuerpo");
}
